竹林中：互动推理 PWA 发布包

当前内嵌剧本版本：2.3.1-visual-deduction-text-cleaned

上传到 GitHub Pages：
1. 打开仓库根目录。
2. 上传本文件夹里的全部内容：
   - index.html
   - manifest.webmanifest
   - sw.js
   - icons/icon-192.png
   - icons/icon-512.png
3. GitHub Pages 设置为 main / root。
4. 等待部署后访问 Pages 链接。

说明：
- 不再需要 story.json；剧本已经内嵌在 index.html 里。
- 旧的 story.json 可以留着，但没有必要。
- 如果更新后网页仍显示旧版，请在链接后加 ?v=新数字，或清除浏览器缓存。
- PWA 安装需要通过 https 访问，GitHub Pages 默认支持。
- 电脑端 Chrome/Edge 可“安装为应用”；手机端可“添加到主屏幕”。
